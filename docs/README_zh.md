# LumaForge 帮助

LumaForge 用于完成一张 RAW 照片，配上你喜欢的风格。文件全程留在你的
设备上，应用只询问必要的色彩细节，并且仅在全分辨率路径能够安全复现结果时
才导出 JPEG。

如果你想使用 RAW 实验室而非构建或扩展它，请从这里开始。

## 指南

- [RAW 实验室指南](./USER_GUIDE_zh.md)：加载 RAW 文件、选择风格、对比并导出。
- [LUT 与导出常见问题](./LUT_AND_EXPORT_FAQ_zh.md)：选择 LUT 合约、理解导出
  禁用状态，以及检查浏览器兼容性。

## 简短版

1. 打开 `/raw` 的 RAW 实验室。
2. 拖入或选择一张 RAW 文件。
3. 等待预览就绪。
4. 选择内置风格或添加 `.cube` LUT。
5. 对比原始和处理后的结果。
6. 当 LumaForge 提示全分辨率路径就绪时，导出 JPEG。

## LumaForge 擅长什么

- 无需安装桌面编辑器即可在 RAW 照片上尝试熟悉的风格。
- 将照片全程留在浏览器本地。
- 使 LUT 输入和输出选择足够清晰，避免常见色彩错误。
- 当源文件和色彩图谱受支持时，导出全分辨率 JPEG。

## 它不是什么

LumaForge 不是目录管理器、批量处理器、云端编辑器或完整的专业调色套件。
它刻意避开账户系统、原生助手、节点图、遮罩、降噪工作流和无限制的
调整堆叠。

如果你需要这些工具，Lightroom、Capture One、DaVinci Resolve 或其他完整
编辑器是更好的选择。LumaForge 刻意做得更聚焦：一张 RAW 文件、一种可控的
风格、一张可信赖的 JPEG。
