# LumaForge Help

LumaForge is for finishing one RAW photo with a look you already like. It keeps
the file on your device, asks only for the color details it needs, and exports a
JPEG only when the full-resolution path can reproduce the result safely.

Start here if you want to use the RAW lab, not build or extend it.

## Guides

- [RAW Lab Guide](./USER_GUIDE.md): load a RAW file, choose a look, compare, and
  export.
- [LUT and Export FAQ](./LUT_AND_EXPORT_FAQ.md): choose a LUT contract, understand
  disabled export states, and check browser compatibility.

## The Short Version

1. Open the RAW lab at `/raw`.
2. Drop or pick one RAW file.
3. Wait for the preview to become useful.
4. Choose a built-in finish or add a `.cube` LUT.
5. Compare the original and processed result.
6. Export a JPEG when LumaForge says the full-resolution path is ready.

## What LumaForge Is Good At

- Trying familiar looks on RAW photos without installing a desktop editor.
- Keeping the selected photo local in the browser.
- Making LUT input and output choices explicit enough to avoid common color
  mistakes.
- Exporting a full-resolution JPEG when the source file and color graph are
  supported.

## What It Is Not

LumaForge is not a catalog, batch processor, cloud editor, or full professional
grading suite. It deliberately avoids account systems, native helpers, node
graphs, masking, denoise workflows, and unlimited adjustment stacks.

If you need those tools, Lightroom, Capture One, DaVinci Resolve, or another
full editor is the better fit. LumaForge is narrower on purpose: one RAW file,
one controlled look, one trustworthy JPEG.
